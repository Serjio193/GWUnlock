# Bundled Third-Party License Texts

This directory contains license texts and notices for third-party components bundled with GWUnlock release binaries.

Some dependency license files depend on the exact binary package used during build, especially OpenOCD and its Windows runtime DLLs. Before publishing a final release, verify the actual files bundled into `GWUnlock.exe` and replace placeholder files with upstream license/notice files from the exact packages used.
