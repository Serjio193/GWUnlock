MinGW Runtime License Notice

GWUnlock bundles OpenOCD for Windows. The OpenOCD runtime currently includes MinGW/GCC runtime DLLs such as:

- libgcc_s_sjlj-1.dll
- libwinpthread-1.dll

The exact MinGW/GCC runtime license notices depend on the OpenOCD binary package used for the release build. Before redistributing a final binary package, verify the original OpenOCD package and include the matching upstream MinGW-w64 / GCC Runtime Library Exception / winpthreads license notices if they are provided with that package.

Do not treat this README as a replacement for the exact upstream runtime license files.
